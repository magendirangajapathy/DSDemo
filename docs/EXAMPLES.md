# Examples & Use Cases

This folder contains compositional examples that demonstrate how components come together to form common application patterns.

Included examples

- `Examples / Dashboard` — A simple admin-style dashboard that composes `NavBar`, `Sidebar`, `Card`, and `Table` components.

How to use

- Open Storybook (`npm run storybook`) and navigate to the `Examples` section to preview patterns.
- Use these examples as starting points for pages; they illustrate layout, spacing, and interaction patterns.

Guidance

- Reuse primitives and tokens from the design system for consistent spacing, color, and typography.
- Keep example pages focused and small — they should illustrate the pattern, not be full applications.
- Link examples from design handoff docs to show designers how components are expected to behave at page-scale.
